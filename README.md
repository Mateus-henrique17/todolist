# To-Do List — React

Aplicação simples de gerenciamento de tarefas desenvolvida com **React**.
O projeto implementa uma lista de tarefas funcional utilizando **estado local** e **persistência no navegador**, sem dependência de APIs externas.

A aplicação permite adicionar tarefas, marcar como concluídas e removê-las. Todas as informações são armazenadas no **localStorage**, garantindo que os dados permaneçam mesmo após recarregar a página.

O projeto também possui **deploy online**, permitindo visualizar a aplicação em funcionamento diretamente no navegador.

---

## Funcionalidades

* adicionar novas tarefas
* marcar tarefas como concluídas
* remover tarefas da lista
* persistência automática utilizando **localStorage**
* interface simples e responsiva

---

## Tecnologias utilizadas

* React
* JavaScript (ES6+)
* CSS
* Vite

---

## Objetivo do projeto

Este projeto foi desenvolvido como prática para:

* manipulação de estado com **React Hooks**
* organização de componentes
* persistência de dados no navegador
* estruturação básica de um projeto React
* deploy de aplicações front-end
